# Skills

A collection of agent skills for Claude Code.

## Available Skills

- **simplify-me** — Relentlessly simplify code, concepts, and decisions. Asks "Is this strictly necessary?" one question at a time, and proposes the simpler version itself.

  ```
  npx skills@latest add ceo0x/skills/simplify-me
  ```

- **why-not** — Challenge every decision in a design or codebase by asking "Why not do it this other way instead?" One alternative at a time, with pros/cons and a final recommendation.

  ```
  npx skills@latest add ceo0x/skills/why-not
  ```
